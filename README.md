# NxJournals Repository
A collection of C# scripts, utilities, and automation tools built for Siemens NX CAD. 
This repository serves as a central place to store, organize, and document working with the NX Open API, including experiments, reusable components, and production‑ready automation. 
Many journals are written as single‑file scripts for easy execution inside NX, while others use the full NXOpen library and may require separate build steps.

This repository exists to:
- Centralize all C# code I’ve written for automating Siemens NX
- Document patterns, workflows, and lessons learned while using NX Open
- Provide reusable utilities for future automation projects
- Track experiments and prototypes as my automation skills evolve
- Serve as a reference for others working with NX Open and C#

Getting Started
Prerequisites

- Siemens NX installed
- Visual Studio Code / Visual Studio (recommended)
- NX Open API enabled (Some modules)
- Access to NXOpen assemblies, typically found in:
    `...\Siemens\NX\NXBIN\`
